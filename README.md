## Pac-Man Game

This is a simple 2D Pacman game that I built using Java. It uses Java Swing and AWT for the graphics and user input.
The player controls Pacman to eat all the food while avoiding four ghosts.

## Gameplay

The goal is simple: Eat all the food dots while avoiding the four colorful ghosts. If a ghost touches you, you lose a life.

1. Use the Arrow Keys (Up, Down, Left, Right) to move Pacman.
2. Eat all the white dots on the screen to score points.
3. Avoid the ghosts. If a ghost touches you, you lose a life.
4. You have 3 lives. If you lose all of them, the game is over.
5. Press any key to restart the game after a Game Over.

## Main Features

1. Movement: Pacman moves smoothly in four directions.
2. Ghosts: There are 4 ghosts that move randomly around the map.
3. Collision Detection: The game checks if Pacman hits a wall, eats food, or touches a ghost.
4. Score System: You get 10 points for every food dot you eat.
5. Map Design: The game level is created using a tile-map system.

## Tools Used
1. Java : The main programming language.
2. Swing: Used to create the game window and panels.
3. AWT  : Used for drawing images and handling keyboard input.

## Project Files(In src)
1. App.java: This is the main file that starts the game and sets up the window.
2. PacMan.java: This file contains the game logic, including how characters move and how the game is drawn.
